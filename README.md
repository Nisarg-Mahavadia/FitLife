# FitLife – Personalized Fitness & Diet Planning App

FitLife is an Android application that helps users plan workouts and maintain a healthy lifestyle. The app provides personalized workout routines, BMI calculation, and diet planning features to support users in achieving their fitness goals.

---

## Screenshots

To check the screenshots of working app open the Screenshots folder

---

## Features

- User authentication using Firebase
- Personalized workout plans
- BMI calculator
- Diet plan suggestions
- Exercise categories (arms, legs, chest, shoulders, core, full body)
- Weekly workout planner
- User profile management
- Firebase Firestore database integration

---

## Technologies Used

- Kotlin
- Android Studio
- Firebase Authentication
- Firebase Firestore
- Material UI Components

## Installation

1. Clone the repository:
git clone https://github.com/Nisarg-Mahavadia/FitLife.git


2. Open the project in **Android Studio**

3. Add your Firebase configuration file:
app/google-services.json

4. Sync Gradle and run the app.
